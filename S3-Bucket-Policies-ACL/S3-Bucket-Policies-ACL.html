Setting up S3 Bucket Policies ACL

  1. Go to S3
  2. Go to a bucket
  3. Click on the Permissions tab
  4. Check the ACL (access control list)
  5. We can add accounts permissions, add bucket policy (JSON or use AWS policy generator),
  6. Click AWS Policy Generator link under Bucket Policy tab page: https://awspolicygen.s3.amazonaws.com/policygen.html
  7. Select S3 bucket policy in dropdown
  8. We're going to creat full public acccess
  9. Principal for everyone is *
  10. You have fine-grained contol but for this example click All Actions
  11. Put in bucket name
  12. Click ADD STATEMENT
  13. Click GENERATE POLICY
  14. Copy and paste JSON into Bucket Policy space on the Bucket Policy Page
  15. SAVE
  16. Note: this is only applied at the BUCKET level, not the Object level - Object level is ACL
