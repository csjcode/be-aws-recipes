RDS PostgreSQL

Setting up RDS PostgreSQL

  1. Go to RDS console: https://us-west-2.console.aws.amazon.com/rds/home?region=us-west-2#launch-dbinstance:ct=GettingStarted:
  2. Check Free Tier only and select PostgreSQL

    3. PostgreSQL: PostgreSQL is a powerful, open-source object-relational database system with a strong reputation of reliability, stability, and correctness. High reliability and stability in a variety of workloads. Advanced features to perform in high-volume environments. Vibrant open-source community that releases new features multiple times per year. Supports multiple extensions that add even more functionality to the database. The most Oracle-compatible open-source database.

  3. Keep all above Settings as default.
  4. Below in Settings, name the instance. ie. testPostgreSQL
  5. Put in admin name (not "admin"): ie. administrator ... and password
  6. Next page - VPC security groups keep as Create New Security Group
  7. If there is an option check "publically available"
  8. Put in a database name ie. sampledb
  9. Keep same port (5432 usually) for "TCP/IP port the DB instance will use for application connections."
  10. Scroll down and change backup retention period to zero
  11. LAUNCH DB INSTANCE and CREATE DATABASE
  12. Go to connect to DB: https://docs.aws.amazon.com/AmazonRDS/latest/UserGuide/CHAP_CommonTasks.Connect.html
  13. Download pgAdmin app:
  14. Put in Name and on the Connection tab "hostname" ad the RDS endpoint: ie. [name].[number].us-west-2.rds.amazonaws.com
  15. Should connect -- if problems then check to make sure instance is "publically accessible"
  16. Check that sampledb database is in there.
