Route 53 - Setup Domain

Setting up a domain with Route 53

  1. Step 1: Obtain a Static URL
    1. EC2 > Elastic IPs > Allocate New Address.
    2. Note the new ip
    3. Associate the new ip with an instance
    4. Verify the new ip is working by typing in a browser
  2. Step 2: Register a Domain Name
    1. Go to Route 53 console view
    2. Register Domain
    3. Find domain you want
    4. Enter contact details
    5. Review agreement
    6. Check email to confirm contact info.
  3. Step 3: Configure DNS
    1. Open hosted zones
    2. Create Record Set button
    3. On the right side of the window, enter www in the Name text box.
    4. Enter the Elastic IP address you created in Value box and Create
    5. Lower TTL for testing
    6. Verify website is working
