EC2 Linux Virtual Machine

Setting up a new EC2 Linux Virtual Machine

  1. Got to EC2 and Launch Instance
  2. Step 1: Select AMI for example: Ubuntu Server 16.04 LTS (HVM), SSD Volume Type - ami-ba602bc2
  3. Step 2: Choose an Instance Type ie. General purpose t2.micro Free tier eligible
  4. Step 7: Review Instance Launch
  5. Get Key pair - Create new key pair, key pair name and download it
  6. Launch instance
  7. SSH to instance
    • Install PuTTY
    • PuTTY does not natively support the private key format (.pem) generated by Amazon EC2.
      • Start PuTTYgen
      • Type of key to generate, choose RSA
      • LOAD the .pem key
      • Save private key to save the key in the format that PuTTY can use
    • Get the ID of the instance
    • Get the public DNS name of the instance
    • Locate the private key, permissions
    • Get the default user name, defaault username is [depends on AMI], for Ubuntu 16 is ubuntu
    • Enable inbound SSH traffic from your IP address to your instance
  8. Log in from SSH with user@publicdns to port 22 with AND Auth .ppk key that was genrated
  9. SSH With Git Bash: ssh -i 'c:\Users\yourusername\.ssh\MyKeyPair.pem' ec2-user@{IP_Address}
