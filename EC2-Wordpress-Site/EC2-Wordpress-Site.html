Setting up Wordpress EC2

  1. Click Launch Instance
  2. AWS Marketplace > WordPress powered by BitNami
  3. Click through Next: Configure Instance Details.
  4. On the Tag screen... enter Name in the Key box and WordPress in the Value box
  5. Launch, Set SSH key
  6. Find the Public IP and put it in the browser - you should get a Wordpress generic template page
  7. To get the Wordpress install password: EC2 management console > WordPress instance > Actions button > Instance Setting > Get System Log
    8. 19.787493] bitnami[1153]: #########################################################################
[   19.796285] bitnami[1153]: #                                                                       #
[   19.807423] bitnami[1153]: #        Setting Bitnami application password to '[password]'         #

[   19.821754] bitnami[1153]: #                                                                       #
[   19.832077] bitnami[1153]: #########################################################################
  8. Go back to your instance IP and add /admin - and then use the password to get into the Wordpress admin console with Username: user
